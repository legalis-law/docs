## Security Features

- **Encryption**: All documents encrypted (AES-256)
- **Access Control**: Role-based permissions
- **Audit Logs**: Track all document access
- **Secure Sharing**: Controlled external sharing
- **Automatic Backups**: Regular encrypted backups

## Access Control

Documents inherit permissions from:
1. Vault type (Team/Private/Knowledge Base)
2. Case team membership
3. Department assignment
4. Explicit shares

<Warning>
  Private vault documents are only accessible to you. Team vault documents follow workspace permissions.
</Warning>
