## Search Features

- **Full-text search**: Search document contents
- **Metadata search**: Filter by type, date, author
- **Tag search**: Find by custom tags
- **Case association**: Search within specific cases
- **Advanced filters**: Combine multiple criteria

## Search Tips

- Use quotes for exact phrases
- Filter by date ranges
- Search within folders
- Use tags for categorization
