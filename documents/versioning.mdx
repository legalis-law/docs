## Version Control

Clerra automatically tracks document versions when you upload updates.

## Features

- Automatic version numbering (v1, v2, v3...)
- Version comparison
- Restore previous versions
- Version notes and change descriptions
- Track who made changes

## Best Practices

- Add version notes explaining changes
- Review version history before major edits
- Archive old versions after finalization
