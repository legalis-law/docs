## Overview

Access LexisNexis legal research database directly from Clerra cases.

## Setup

1. Go to Settings > Integrations
2. Select **LexisNexis**
3. Enter your LexisNexis API credentials
4. Test connection
5. Save integration

## Features

- Research from case context
- Save research to case notes
- Cite checking
- Case law updates
- Statute search

<Note>
  Requires active LexisNexis subscription and API access.
</Note>
