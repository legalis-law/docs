## Enabling Integrations

<Steps>
  <Step title="Access Settings">
    Click workspace name > Settings > Integrations tab
  </Step>

  <Step title="Select Integration">
    Choose the integration to enable
  </Step>

  <Step title="Configure">
    Enter API keys or OAuth credentials
  </Step>

  <Step title="Test Connection">
    Verify integration is working
  </Step>

  <Step title="Save">
    Integration is now active for your workspace
  </Step>
</Steps>

<Note>
  Only Super-Admins can manage integrations. Integration credentials should be kept secure.
</Note>
