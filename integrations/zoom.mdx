## Overview

Integrate Zoom for client meetings, depositions, and team collaboration directly from Clerra.

## Setup

1. Go to Settings > Integrations
2. Select **Zoom**
3. Click **Connect to Zoom**
4. Authorize Clerra to access your Zoom account
5. Integration is ready

## Features

- Schedule meetings from case calendar
- Automatic meeting links in event details
- Recording storage in case vault
- Client portal meeting access

<Tip>
  Use Zoom for secure client consultations and remote depositions.
</Tip>
