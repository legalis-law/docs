## Creating Custom Playbooks

Build playbooks tailored to your firm's processes.

## Steps

1. Perfect a case workflow
2. Document the checklist items
3. Save as playbook template
4. Add description and tags
5. Set visibility (Private/Department/Workspace)

<Note>
  Requires appropriate permissions to create workspace-level playbooks.
</Note>
