## Comments

Add comments to cases, documents, and tasks for team collaboration.

## Features

- **@Mentions**: Notify specific users
- **Threading**: Reply to comments
- **Attachments**: Add files to comments
- **Rich Text**: Format comments with markdown
- **Reactions**: React with emojis

<Tip>
  Use @mentions to notify team members about important updates.
</Tip>
